# Universal File Processor (UFP)

Universal File Processor یک ابزار ماژولار برای تبدیل انواع فایل‌ها با رابط کاربری زیبای کنسولی (Rich/Questionary) است.

## قابلیت‌ها
- تبدیل ویدیو/صدا با FFmpeg
- پردازش تصویر با Pillow / ImageMagick
- تبدیل اسناد با Pandoc
- تبدیل فایل‌های Office با LibreOffice
- تبدیل کتاب‌های الکترونیک با Calibre
- رابط کاربری تعاملی، اعلان‌های زیبا، گزارش‌گیری

## پیش‌نیازها
- Python 3.10+ (تست‌شده روی 3.14)
